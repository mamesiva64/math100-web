# 百ます計算 採点Webアプリ MVP

サーバ不要で動く、百ます計算の問題生成・印刷・タイマー・手動採点・撮影セル切り出し・履歴表示をまとめたMVPです。

## 使い方

`index.html` をブラウザで開くと動作します。iPhoneではSafariでホーム画面追加するとPWAとして利用できます。

## 実装済み

- 10x10足し算問題生成
- A4縦の印刷用紙、四隅マーカー、固定マス
- タイマー
- 100問の手入力採点
- IndexedDB保存
- 履歴一覧、直近10回グラフ
- カメラ/画像アップロード
- マーカー検出を使った固定レイアウトの100セル切り出し
- 切り出し画像付きの確認、手修正、保存

## 数字分類について

`src/app.js` の `classifyCell` が数字分類の差し替え境界です。現状はONNXモデル未配置のMVP挙動として、空欄検出のみ行い、数字がありそうなセルは低信頼として要確認へ回します。

`models/digit_classifier.onnx` と `onnxruntime-web` を組み込む場合は、同関数で28x28などモデル入力形式へ正規化して推論してください。
